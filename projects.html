VitalDb VitalDB-Monitor

This project explores the analysis and visualization of vital signs using VitalDB, an open-source database for physiological monitoring. The focus is on two critical signals: arterial pressure waves and plethysmography waves. The project investigates key parameters of these signals and examines their interrelationship. To present the findings, an interactive dashboard was developed that provides intuitive visualizations of the analysis. In addition, the dashboard features a live monitoring component, displaying real-time blood pressure and heart rate from streaming data, similar to a clinical patient monitor.

github : https://github.com/eshita53/VitalDB-Monitor.

Anomaly Detection

This project focuses on analyzing pump sensor data to develop an anomaly detection model capable of identifying abnormal behavior. By detecting irregular patterns in the data, the model enables early problem identification and helps prevent potential system failures.

Github: https://github.com/eshita53/Unsupervised-learning/tree/main/Anomaly_detection#anomaly-detection

Identifying Biological Substitutes for Synthetic Compounds Using Clustering

Synthetic products can be harmful to health and the environment, and there is growing interest in replacing them with natural alternatives. In this project, we analyzed two clustering algorithms to identify natural compounds that are structurally similar to synthetic compounds, aiming to find potential substitutes.

Github: https://github.com/eshita53/Unsupervised-learning/tree/main/Anomaly_detection#anomaly-detection

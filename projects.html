VitalDB Monitor

This project explores the analysis and visualization of vital signs using VitalDB, an open-source database for physiological monitoring. The focus is on arterial pressure waves and plethysmography waves. An interactive dashboard was developed for intuitive visualizations and live monitoring of real-time blood pressure and heart rate.

Anomaly Detection

This project analyzes pump sensor data to develop an anomaly detection model capable of identifying abnormal behavior. It enables early problem identification and helps prevent potential system failures.

Identifying Biological Substitutes for Synthetic Compounds Using Clustering

Synthetic products can be harmful to health and the environment. This project analyzes clustering algorithms to identify natural compounds structurally similar to synthetic compounds, aiming to find potential substitutes.
